Computers languages

Computer simply a machine. It cannot do anything on its own. If we want to perform any action on the Computer, it needs instructions accordingly. Since computer does not understand natural languages which are developed, a computer language is means by which instructions and data ae transit to computer. Computer understand instructions written in these languages. These can be easily translated into a machine level languages, which a computer understands.

Computers languages are divided into the following types:

  • Low Level Languages[LLL]
  • High Level Languages[HLL]
  • Fourth Generation Languages[4GL]
